# DevEx | Micro-SaaS Marketplace

**DevEx** is a high-performance marketplace designed for buying and selling Micro-SaaS projects. Built with a focus on **System Design**, **Security**, and **Scalable Architecture**, this project implements backend concepts used in real-world production environments.

Developed as a core project for the **Data Structures (3rd Semester)** curriculum at **UBIT, University of Karachi**.

---

## 🛠️ Tech Stack

| Layer | Technology |
| :--- | :--- |
| **Frontend** | React.js (Vite), Tailwind CSS |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB (NoSQL) |
| **Authentication** | OAuth 2.0 (Google), JWT, Middleware Protection |
| **Security** | Rate Limiting, Modular MVC Architecture |

## ✨ Key Features

* **Production-Grade Security:**
    * **Rate Limiting:** Protects against Brute Force and DoS attacks using `express-rate-limit`.
    * **Secure APIs:** Implementation of Helmet.js and CORS for standardized web security.
* **Advanced Auth:** Hybrid system featuring Google OAuth 2.0 and encrypted JWT-based sessions.
* **Scalable Architecture:** A strict MVC (Model-View-Controller) pattern to ensure the codebase remains maintainable as the project grows.

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/salmanzulfiqar/DevEx.git
cd DevEx
```

### 2. Setup Backend
```bash
cd Backend
npm install
npm run dev # Starts on http://localhost:3000
```

### 3. Setup Frontend
```bash
cd FrontEnd
npm install
npm run dev # Starts on http://localhost:5173
```
